Rangkuman Chapter 2 Discovering Computers

Internet jg disbt Net adalah kumpulan jaringan di dunia yg menghubungkan jutaan orang ataupun instansi. Internet berasal sbg ARPANET yg memiliki tujuan memungkinkan para ilmuan yg berbeda untk berbagi informasi dan bekerja sama. Host/server adlh komputer yg memberikan layanan dan sambungan ke komputer-komputer lain pd suatu jaringan. EVOLUSI INTERNET. Internet2 yaitu menghubungkan lebih dari 200 universitas dan 115 perusahaan melalui jaringan berkecepatan tinggi swasta; menghubungkan ke internet contohnya kabel layanan internet, DSL, FTTP, telepon tetap nirkabel, jaringan radio selular, Wi-Fi, satelit layanan Internet. Akses ke internet gratis atau untuk biaya :ISP :menyediakan akses Internet untuk wilayah geografis tertentu; OSP :Memiliki banyak anggota-satunya fitur;WISP: Menyediakan akses internet nirkabel untuk komputer dan perangkat mobile. Alamat internet terdiri dari : Alamat IP yaitu angka yang secara unik mengidentifikasi setiap komputer atau perangkat yang terhubung ke Internet; Nama domain adalah versi teks dari alamat IP;Server DNS menerjemahkan nama domain ke alamat IP-nya terkait.

World, Wide Web atau Web adalah layanan yang banyak dimanfaatkan dalam internet, terdiri atas kumpulan dokumen elektronik diseluruh negara. Sebuah situs web adalah kumpulan halaman web yang saling berhubungan denga hal-hal liannya yang terkait. Server web adalah komputer yang memberikan halaman web yang diminta ke komputer Anda. Penjelajah web atau browser adalah perangkat lunak aplikasi yang dapat digunakan untuk mengakses dan melihat halaman web,contohnya internet explorer, firefox, opera, safari, google chrome. Mengunduh adalah proses komputer menerima informasi. Alamat web, setiap halaman web memiliki alamat yang disebut URL. Sebuah alamat web biasa di awali dengan http atau Hypertext Transfer Protocol yaitu sekelompok aturan yg mendefinisikan bagaimana halaman web ditransfer ke internet. Navigasi alamat web, hampir semua halaman web mengandung hyperlink, adalah sambungan bawaan ke halaman web lainnya atau bagian sama dr situs web yg menampilkan informasi terkait. Mencari informasi di web, memiliki 2 type: direktori subjek yt pengelompokan halamanweb dengan kategori yang sudah ditentukan dan mesin pencari yt program yang melakukan pencarian situs dr halaman web. Jenis-jenis situs web : portal, situs web yg menawarkan layanan internet dari suatu lokasi; berita,situs yg mengandunga artikel yang terkait dengan peristiwa; informasi, situs informasi yg mengandung informasi yg sifatnya faktual; bisnis, situs bisnis yg menampilkan dan menawarkan barang dan jasa; pendidikan, situs pendidikan yg menyediakan metode mengajar sec formal dan internal; hiburan, situs yg menawarkan segala hal yg menarikdan interaktif; advokasi, situs yg menjelaskan suatu tujuan,pendapat atau gagasan; blog, situs yang terdiri dari artikel-artikel kepribadian dari penulis; wiki, situs web yg memungkinkan pengguna untuk menghapus, menambah, dan mengubah isi dari situs tsbt; onlienejaringan sosial adalah sebuah situs web yang mendorong anggota dalam komunitas online untuk berbagi minat mereka seperti, ide-ide, cerita,dll. Bagian dari multimedia web adalah grafik, animasi, audio, video,virtual reality, dan plug-in. E-commerce(electonic commerce),merupakan transaksi bisnis yg terjadi dlm jaringan elektronik seperti internet.

Layana Internet lainnya seperti :E-mail(surat elektronik),penyampaian pesan dan file melalui suatu jaringan komputer; FTP(File Transfer Protocol),suatu standart internet yg dpt mengupload dan mendownload file dengan komputer lain di internet; Mailling list,kelompok dari nama dan alamat e-mai lyg memiliki suatu nama tunggal; Pesan instan, layanan komunikasi internet yg dpt memberitahu Anda ketika orang lain online,dan Anda dpt bertukar file/ngobrol secara pribadi; Ruang ngobrol, lokasi pd server intrnet dimana orang-orang dapat berbicara satu sama lain; VoIP(Voice over IP),memungkinkan pengguna dpt berbicara dengan pengguna lainnya; kelompok warta dan papan pesan, area online di mana para pengguna melakukan diskusi tertulis mengenai subjek tertentu.

Netiquette (etika di internet) adalah kode perilaku yang dapat diterima yang harus diikuti oleh para pengguna internet, artinya perilaku yang diharapkan dari semua pengguna online.
